Features

Learn about the features of Cantiga

Projects, groups and areas

Cantiga supports multiple projects, which can be further divided into groups and areas. Each of it has its own intranet workspace, where the members can meet and work. Leaders can request creating the new areas, which then undergo a moderation process before being approved for the project. This model works for small projects, and allows them growing in the future to support even country-wide projects.

Project customization

You can customize projects to a deep level, by enabling and disabling individual modules, configuring their behavior, customizing area profile fields, and even changing the application texts, and e-mail templates.

Statistics

Statistics and charts provide a general overview of the project situation.

On-line course module

Provide on-line courses for your members, create exam tests and verify the training progress.

Milestone module

Milestones are the key points that your projects, groups and areas must reach. You can use them to specify the main goals, and control who achieved them.

Discussion module

Discussions between members are possible with a chat-like discussion module that supports multiple channels and individual discussions for each group or area.

Multilingual

Cantiga is currently translated into English and Polish, and it is easy to provide a new translation.

Modular design

Even the core functionality is provided as a set of modules. You can easily extend Cantiga with new features by writing new ones.

Rock-solid technology

Cantiga is built with Symfony 3 application framework, the most mature platform for constructing applications available for PHP language.

Cantiga is being actively developed since October 2015. More features are expected to come!
