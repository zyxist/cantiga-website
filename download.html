How to get Cantiga

Stable branch

The current stable branch is 1.0.x. We recommend using the most recent stable branch for production purposes. The versions released on this branch contain bugfixes only.

Branch name for syncing with Git repositories: 1.0.x

Clone from Git:

git clone -b 1.0.x https://github.com/zyxist/cantiga.git

Development branch

The development branch is used for implementing new features and backward-incompatible changes. We do not recommend using this version for production purposes, as the commits may break things without warnings.

Clone from Git:

git clone -b master https://github.com/zyxist/cantiga.git
