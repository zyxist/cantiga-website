Get help

What to do in case of problems

Discussion group

Post your question on our discussion list. The list is for Cantiga-related posts, and please remember about using English language.

Gitter chat

If you have a GitHub or Twitter account, you can catch us at our Gitter chat channel. Even if we are not present at the given time, we'll be able to answer you later.

GitHub

If you are a developer and own a GitHub account, you can visit the project Github profile.
